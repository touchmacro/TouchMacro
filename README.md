# 📱 터치 매크로 편집기 사용자 매뉴얼

본 매뉴얼은 매크로의 각 동작(Action)을 설정하고 편집할 때 사용하는 기능들에 대한 안내입니다. 모든 동작에는 사용자가 알아보기 쉽도록 **이름(Name)**을 지정할 수 있습니다.

## 1. 👆 기본 동작 및 제어 (Basic Actions)
매크로의 가장 기초적인 터치 및 대기 동작을 설정합니다.

* **Touch Position (터치 좌표):** 화면에서 실제로 터치(또는 스와이프)할 위치의 좌표(X, Y)를 설정합니다.
* **Gesture Duration (동작 지속 시간):** 스와이프 또는 프레스(길게 누르기) 시 해당 동작을 얼마 동안 유지할지 초 단위로 설정합니다.
* **Delay Time (대기 시간):** 이 단계를 실행하기 전에 설정한 시간(초 단위)만큼 일시 정지하며 대기합니다.

---

## 2. 👁️ 화면 인식 및 조건 설정 (Screen Recognition)
화면의 이미지나 텍스트를 인식하여 상황에 맞는 다음 동작을 결정합니다.

### 이미지 매칭 (Image Matching)
* **Search Area (검색 범위):**
    * **Fixed position:** 지정된 고정 영역에서만 이미지를 비교합니다.
    * **Full screen / Custom Area:** 화면 전체 또는 지정 영역에서 이미지를 검색합니다.
    * **Grayscale (흑백 스캔):** 화면을 흑백으로 스캔하여 검색 속도를 향상시킵니다.
* **Similarity Threshold (일치도 기준):** 실제 화면이 타겟 이미지와 얼마나 똑같아야 성공으로 인정할지 정확도 기준(예: 80%)을 설정합니다.
* **Scan Interval (스캔 주기):** 이미지를 확인하는 주기를 설정합니다. 빠르게 변하는 화면을 감지하려면 속도를 높이세요.
* **Search Duration (최대 검색 시간):** 이미지를 찾을 때까지 대기하는 최대 시간입니다. 
    * **0으로 설정하면 무한 탐색합니다.**
    * **※ 주의:** 결과에 따라 다른 단계로 이동하는 '조건부 동작(If Found / If Not Found)'을 사용하려면 반드시 **0보다 큰 값**을 입력해야 합니다.
* **If Found / If Not Found (찾았을 때 / 찾지 못했을 때):** 이미지를 찾았거나 찾지 못했을 경우 각각 이동할 다음 매크로 단계 번호를 지정합니다.

### 텍스트 및 숫자 인식 (Text & Range)
* **Text / Range:** 화면의 글자(텍스트)를 인식하여 일치하는 내용을 찾거나, 숫자를 인식해 특정 범위 내에 포함되는지 확인합니다.
* **Confidence Score Threshold (최소 신뢰도 기준):** 텍스트 인식 엔진이 문자를 읽어낼 때 성공으로 인정할 확신 점수를 설정합니다. 값이 높을수록 더 정확하게 판독된 글자만 찾아냅니다.

---

## 3. ⚙️ 시스템 동작 (System Actions)
안드로이드 기기의 기본 시스템 기능을 매크로로 제어합니다.

* **Action Type (동작 유형):** 아래와 같은 기기 시스템 명령 중 하나를 선택하여 실행합니다.
    * **Screenshot:** 현재 화면을 캡처합니다.
    * **Home:** 홈 화면으로 이동합니다.
    * **Back:** 이전 화면으로 돌아갑니다.
    * **Recent:** 최근 실행한 앱 목록을 띄웁니다.

---

## 4. 🎯 터치 고급 설정 (Touch Action)
정교한 매크로 작동을 위한 세부 터치 옵션입니다.

* **Skip Touch (터치 생략):** 실제 터치 동작을 생략합니다. 대기 시간 부여나 이미지 검색 여부만 체크할 때 유용합니다.
* **Double Click (더블 클릭):** 해당 위치를 연속으로 두 번 빠르게 터치합니다.
* **Tap Offset (터치 위치 이동):** 감지된 이미지의 위치가 아닌, 지정한 거리만큼 떨어진 다른 곳을 터치하도록 설정합니다.

---

## 5. 🛡️ 탐지 회피 (Anti-Detection)
사람이 직접 조작하는 것처럼 보이게 하여 기계적인 패턴을 숨깁니다.

* **Random Coordinates (랜덤 좌표):** 터치 위치를 오차 범위 내에서 무작위로 변경하여 동일한 좌표만 반복 터치하는 것을 방지합니다.
* **Random Delay (랜덤 대기 시간):** 기본 대기 시간에 무작위 시간을 더해 매크로의 실행 간격 패턴을 불규칙하게 만듭니다.

---

## 6. 🔄 흐름 제어 및 서브 매크로 (Flow Control)
반복 횟수 제어 및 매크로 간 연결을 통해 복잡한 자동화를 구현합니다.

### 루프 카운터 (Loop Counter)
매크로 실행 횟수를 계산하여 특정 횟수마다 다른 단계로 이동(분기)시킵니다.
* **Target Count (목표 횟수):** 실행 횟수가 이 숫자에 도달하면 지정한 단계로 이동합니다.
* **Default Jump (기본 이동 단계):** 설정한 조건(Rule)에 해당하지 않을 때 기본적으로 이동할 단계를 지정합니다.

### 카운터 초기화 (Reset Counter)
* **Target Counter (초기화할 카운터):** 실행 횟수를 0으로 되돌릴 카운터가 위치한 매크로 단계 번호를 입력합니다.

### 서브 매크로 실행 (Run Sub-Macro)
* **Select Macro (매크로 선택):** 실행할 다른 매크로 파일을 불러옵니다.
* **Return When Finished (실행 후 돌아오기):** 불러온 매크로가 끝나면 다시 현재 매크로로 돌아와 다음 작업을 계속합니다.

---

## 7. 🔔 상태 알림 (Status Alerts)
매크로 진행 상황을 진동, 소리, 팝업 메시지로 사용자에게 알립니다.

* **Vibrate (진동):** 알림 발생 시 기기에 진동을 울립니다.
* **Sound (소리 재생):** 지정한 알림음 파일을 재생합니다.
* **Alert Duration (알림 지속 시간):** 팝업이 유지되는 시간을 설정합니다. (0을 입력하면 사용자가 직접 닫을 때까지 유지됩니다.)
* **Show Pop-up (팝업창 표시):** 화면에 안내 메시지 창을 띄웁니다.
    * **Message Text (메시지 내용):** 팝업창에 표시할 문구를 입력합니다.
